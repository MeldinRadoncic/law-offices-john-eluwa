# Responsive Design Requirements
The website MUST BE FULLY RESPONSIVE across all device sizes:
- **Mobile phones (sm, md)**: MUST BE PERFECT - Title, description, and primary CTA button must be visible above the fold (no scrolling needed for first screen)
- **Tablets (iPad, large iPad)**: Full responsive layout
- **Laptops (13", 14", 15" screens)**: Optimal layout and spacing
- **Large screens (lg, xl, 2xl)**: Beautiful scaling with proper spacing

Mobile optimization is CRITICAL - the first impression on mobile devices must be PERFECT, modern, and fully responsive with all key content visible without scrolling.
